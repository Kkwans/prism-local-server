## 修复的问题

### 1. ✅ build.bat 中文乱码问题

**问题描述**：
- 运行 build.bat 时终端输出中文显示为乱码
- 例如："鎵撳寘鑴氭湰" 而不是 "打包脚本"

**根本原因**：
- Windows CMD 默认使用 GBK 编码
- 批处理文件没有设置 UTF-8 编码

**解决方案**：
- 在 build.bat 第二行添加 `chcp 65001 >nul`
- 强制使用 UTF-8 编码显示中文

**验证方法**：
```bash
cd prism-server-flet
build.bat
# 应该看到正确的中文输出
```

### 2. ✅ EXE 无法运行 - flet_desktop 模块缺失

**问题描述**：
- 双击 PrismLocalServer.exe 弹出错误
- 错误信息：`ModuleNotFoundError: No module named 'flet_desktop'`
- 错误信息：`NameError: name 'exit' is not defined`

**根本原因**：
- PyInstaller 打包时没有包含 Flet 的所有依赖模块
- Flet 需要特殊的打包配置才能正常工作
- 缺少 flet_core、flet_runtime 等核心模块

**解决方案**：

更新 build.spec 文件，添加：
1. 导入 PyInstaller 的 collect_data_files 和 collect_submodules
2. 收集 Flet 的所有数据文件：`flet_datas = collect_data_files('flet')`
3. 收集 Flet 的所有子模块：`flet_hiddenimports = collect_submodules('flet')`
4. 在 hiddenimports 中添加：
   - flet.core
   - flet.utils
   - flet_core
   - flet_runtime
   - PIL._imaging

**验证方法**：
```bash
cd prism-server-flet
build.bat
# 等待打包完成
cd dist
PrismLocalServer.exe
# 应该能正常启动应用
```
