# Prism Local Server - 项目总结

## 项目概述

Prism Local Server 是一个基于Flet (Flutter) 的现代化前端静态文件部署工具，用于快速将HTML静态文件部署到本地HTTP服务器，支持局域网访问。

## 技术选型

### UI框架: Flet 0.82+
- **选择理由**: 
  - UI颜值天花板（Flutter的Material Design 3）
  - 60/120fps流畅动画
  - 完美的跨平台支持
  - AI友好的声明式API
  - 打包简单（独立EXE）

### HTTP服务: Python http.server
- **选择理由**:
  - Python标准库，无需额外依赖
  - 支持多线程（ThreadingHTTPServer）
  - 易于扩展和定制

### 系统托盘: pystray
- **选择理由**:
  - 轻量级，跨平台
  - 简单易用的API
  - 支持自定义图标和菜单

## 技术亮点

### 1. 配置文件独立性
配置文件存储在用户目录而非EXE所在目录，确保：
- EXE可以复制到任何位置运行
- 多用户环境下配置隔离
- 符合Windows应用最佳实践

### 2. 资源加载性能
- 使用类变量传递部署目录，避免参数传递问题
- 支持HTTP Range请求，视频可随意拖动
- 启用TCP_NODELAY，减少网络延迟
- 8KB缓冲区，优化大文件传输

### 3. 端口管理策略
- 默认端口9000（避免8888权限问题）
- 端口被占用时自动递增
- 支持手动配置默认端口
- 端口范围验证（1024-65535）

### 4. 多服务管理
- 每个服务独立线程运行
- 服务状态实时监控
- 运行时长精确统计
- 优雅的服务停止机制
