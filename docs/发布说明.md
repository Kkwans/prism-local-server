## 升级指南

### 从 v1.0.0 升级

1. **备份配置**（如果需要）
   - 旧版本配置在: `prism-local-server/config/`
   - 新版本配置在: `C:\Users\{username}\.prism-server\`

2. **卸载旧版本**
   - 删除旧的EXE文件
   - 可选：删除旧的配置文件

3. **安装新版本**
   - 下载 `PrismLocalServer.exe`
   - 放置到任意目录
   - 双击运行

4. **重新配置**
   - 首次运行会自动创建默认配置
   - 根据需要调整设置

## 下载

### EXE文件
- 文件名: `PrismLocalServer.exe`
- 大小: ~11MB
- 位置: `dist/PrismLocalServer.exe`

### 系统要求
- Windows 11 (22H2或更高版本)
- 无需安装Python或其他依赖

## 使用建议

### 最佳实践
1. 首次使用前先打开设置，确认配置
2. 部署大量文件时，建议关闭自动打开浏览器
3. 多服务运行时，注意端口号避免冲突
4. 定期查看日志文件排查问题

### 性能建议
1. 单个目录文件数建议 < 5000
2. 同时运行服务数建议 < 5个
3. 大文件（>100MB）加载可能较慢
