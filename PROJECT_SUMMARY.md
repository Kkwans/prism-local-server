# Prism Local Server - 项目完成总结

## 项目概述

Prism Local Server（棱镜本地服务器）是一个Windows 11桌面应用程序，用于快速将HTML静态文件部署到本地HTTP服务器，支持本地访问和局域网访问。

**项目状态：** ✅ 核心功能已完成，可正常使用

**开发时间：** 2026-03-15

**Git仓库：** git@github.com:Kkwans/prism-local-server.git

## 使用说明

### 开发环境运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

### 打包为EXE

1. 使用打包脚本（推荐）：
```bash
build.bat
```

2. 手动打包：
```bash
pyinstaller build.spec --clean --noconfirm
```

### EXE使用

1. 进入打包目录：`dist/PrismLocalServer/`
2. 双击运行：`PrismLocalServer.exe`
3. 选择要部署的目录
4. 点击"启动服务"按钮
5. 浏览器自动打开访问地址
