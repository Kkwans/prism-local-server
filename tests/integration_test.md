### 测试 4: 相对路径资源加载

**测试步骤：**
1. 创建测试目录结构：
   ```
   test-dir/
   ├── index.html
   ├── css/
   │   └── style.css
   ├── js/
   │   └── script.js
   └── images/
       └── logo.png
   ```
2. 在 index.html 中使用相对路径引用资源：
   ```html
   <link rel="stylesheet" href="css/style.css">
   <script src="js/script.js"></script>
   <img src="images/logo.png">
   ```
3. 启动服务并在浏览器中访问
4. 验证所有资源都正确加载

**预期结果：**
- 所有 CSS、JS、图片资源都正确加载
- 浏览器开发者工具中没有 404 错误
- MIME 类型正确（text/css, application/javascript, image/png）

### 测试 8: 视频 Range Requests 支持
